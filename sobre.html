Sobre o Projeto

Objetivo

Plataforma de análise de imagens online, construída com tecnologias web padrão HTML, CSS e JAVASCRIPT, que permite aos usuários submeter imagens e obter feedback imediato sobre os objetos, cenas e rostos presentes nelas. A solução garante que as análises visuais sejam tanto precisas quanto rápidas, com uma robusta segurança integrada para proteger os dados dos usuários e as imagens enviadas.
